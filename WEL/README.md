## Événement DNS Server - ID 4
   
Détails de l’événement  
  - Source : DNS Server Service (Microsoft-Windows-DNS-Server-Service)  
  - ID : 4  
  - Niveau : Information (Level 4)  
  - Nom : DNS_EVENT_ZONE_LOAD_COMPLETE  

Signification  
  
Cet événement est généré lorsqu’une zone DNS a terminé son chargement avec succès par le service serveur DNS. Il peut s’agir :  
    - du chargement initial de la zone au démarrage du service,  
    - d’un rechargement déclenché manuellement ou après modification de la zone.  
  
L’événement confirme que les données de la zone sont disponibles et que le serveur peut répondre aux requêtes pour cette zone.
